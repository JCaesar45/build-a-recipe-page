# Classic Chocolate Chip Cookies

A simple HTML page displaying a recipe for classic chocolate chip cookies.

## Preview

![Classic Chocolate Chip Cookies](https://cdn.freecodecamp.org/curriculum/labs/recipe.jpg)

## Project Description

This project is a basic static HTML page showcasing a delicious recipe for soft, chewy chocolate chip cookies. It demonstrates the use of common HTML elements such as headings, paragraphs, lists, and images.

## Features

- **Title and description** of the recipe
- **Ingredients** listed using an unordered list
- **Instructions** listed using an ordered list
- **Image** of the final cookies

## Code Structure

- `<!DOCTYPE html>`: Declares the HTML document type
- `<html lang="en">`: Sets the language to English
- `<head>`: Contains meta information and the page title
- `<body>`: Contains the visible content:
  - `<h1>` for the main recipe title
  - `<p>` for the description
  - `<h2>` for section headings (Ingredients and Instructions)
  - `<ul>` and `<li>` for listing ingredients
  - `<ol>` and `<li>` for listing steps
  - `<img>` to display an image of the cookies

## How to Use

1. Clone or download the project.
2. Open the `index.html` file in any web browser.
3. View the recipe and enjoy baking!
